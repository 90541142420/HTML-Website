HTML Tags

<h1> - <h6>
Defines headings of different levels where <h1> is the highest level and <h6> is the lowest.
<p>
Defines a paragraph of text.
<a>
Defines a hyperlink, which can link to another web page, a file, an email address, etc.
<img>
Embeds an image into the HTML document.
<ul> / <ol> / <li>

<ul> defines an unordered list, <ol> defines an ordered list, and <li> defines list items in both types of lists.
